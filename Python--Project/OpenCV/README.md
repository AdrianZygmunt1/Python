# 🛠 Wymagania i Instalacja
Aby uruchomić ten projekt, musisz mieć zainstalowanego Pythona oraz następujące biblioteki. Możesz je zainstalować za pomocą menedżera pakietów pip:

```bash
pip install opencv-python mediapipe numpy

```
# Opis użytych bibliotek:
OpenCV (cv2): Odpowiada za obsługę kamery i przetwarzanie obrazu.

MediaPipe (mp): Wykorzystuje modele uczenia maszynowego do śledzenia dłoni/ciała w czasie rzeczywistym.

NumPy (np): Służy do szybkich obliczeń na macierzach (obrazach).
